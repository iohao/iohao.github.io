2022-07-06 - v17.1.1​

ioGame 上传到中央仓库中。

ioGame 版本规则 x.y.z

  • x 表示当前使用的 JDK 版本
  • y 表示 ioGame API 变更版本 （但基本上不会变动，常规下是变动 x 才会变动 API）
  • z 表示 ioGame 新特性、新功能、新模块、bugfix 相关

ioGame 的 x 会跟着最新的 JDK LTS 版本来走的，目的是确保 ioGame 的 API 不会发生很大的变化。 为了保持新活力与接受新鲜事物， ioGame 基本会用上最新的 JDK LTS，也就是说，下一个 x 将会是 21。 x 一般延后 24 个季度，给开发者一个缓冲，即下一个 JDK LTS 出来后，那么 ioGame 的 x 会在 24 个季度后跟上。
